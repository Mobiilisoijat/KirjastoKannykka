# KirjastoKännykkä

Kirjastokännykkä is an mobile app for different library needs, having useful features for readers. Kirjastokännykkä encourages new users to read books with social features to see how many books one may have read.

Our application uses Finna's services to make searches for books. These books can be searched using a regular search or if you are not willing to write your book's name, using the book's ISBN instead (Lue ISBN).

Kirjastokännykkä has a readinglist (lukulista) to keep you on track, which books you have read. By making a search and selecting your desired book, you see more details of the book. You can then add the book to the reading list as 'completed', 'reading' or 'planning'. You may also leave a review on the book from this page.

Readers can enter Finnish libraries with our application! (allegedly) Using a QR-code generated for the user, you may enter a library (allegedly) by showing it to the reader.

If our users are lonely our companion KirjaBotti may soothe you. With this chatBot you can ask questions about books, generate images or even play a quiz-game with. 

Users can see different statistics of since when they have been shortsighted. Statistics display how many books the reader has read, how many each year, their readinglist status and how many books have been favourited.  Reader can reflect on their compassion to reading using these features.

Unsatisfied users may leave user feedback on our application. Our developers will respond eagerly to any good suggestions.




## If you want to use many of our social features, you need to register a new account to our application. 

Features that require a logged in user are 
* leaving reviews on books
* inspecting and adding books to reading list
* book statistics
* Library access via QR-code
